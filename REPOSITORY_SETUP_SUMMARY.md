## 📝 Before First Commit

**CRITICAL: Check for sensitive data**

Run this to see what will be committed:
```powershell
git status
```

**DO NOT COMMIT these files if they appear:**
- `firebase-service-account.json`
- `.env` or `.env.*`
- `application.properties` (with real API keys)
- Any file containing passwords or API keys

They should already be in `.gitignore`, but double-check!
